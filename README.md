# gArch64
A hypothetical idea of a 64-bit processor, Although only the 32-bit baseline is implemented as of now

### The Emulator
It emulates the system

See [its documentation](emulator/doc.md) for info on how to boot your disk image and how to debug your boot program

### The Assembler
Convert Assembly code into executable binary file

See [its documentation](assembler/doc.md) for the system's working and how to write code for the architecture
